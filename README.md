# About
![screenshot of the app](/images/screenshot.png)
An example of Web Audio API integration with 3rd party APIs—in this case the Visual Crossing Weather API from RapidAPI. See the page at [https://ogbabydiesal.github.io/weather-song/](https://ogbabydiesal.github.io/weather-song/).

Enter a location and experience a fun "pop-up!"

If your location's tempature is =< 32 degrees...You'll see this with a fun toon to listen to!
![image alt](https://github.com/syc9459-ui/API-Website/blob/main/frozen_guy.jpeg?raw=true)

If your location's tempature is >32, but less than 75, this will pop-up with a another fun toon!
![image alt](https://github.com/syc9459-ui/API-Website/blob/main/sun_baby.jpeg?raw=true)

If your location's tempature is >75, then this will pop-up with, yet again, another fun toon!
![image alt](https://github.com/syc9459-ui/API-Website/blob/main/heat_miser.jpeg?raw=true)
